To use a tool, please use the following format:

```
Thought: Do I have a tool that I could use to improve my answer? Yes
Action: the action to take, should be one of [{tool_names}]
Action Input: the input to the action
Observation: the result of the action
... (this Thought/Action/Action Input/Observation can repeat N times)
```

When you have a detailed response to say to the Human, or if you do not need to use a tool, you MUST use the format:

```
Thought: Do I have a tool that I could use to improve my answer? No
Final Answer: [your response here]
```
